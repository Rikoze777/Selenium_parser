# Selenium_parser

Парсер данных с сайта `https://www.nseindia.com/`. После сбора данных данный скрипт выполняет дополнительные манипуляции на сайте для имитации пользователя.

## Установка

```bash
pip install -r requirements.txt
```

## Использование

```bash
python3 parser.py
```
